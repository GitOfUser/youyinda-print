# 优印达 YouYinDa — Stitch 界面设计提示词

> 本提示词用于 **Stitch（AI 界面设计工具）**，请基于以下内容生成优印达微信小程序与后台管理端的完整高保真界面设计。

---

## 一、项目概述

优印达（YouYinDa）是一个**集云打印与快递寄件于一体的微信小程序聚合平台**。平台对接第三方服务商 API，为用户提供：

- **云打印**：上传文件 → 配置打印参数 → 在线支付 → 第三方生产配送
- **快递寄件**：填写寄收件信息 → 比价选公司 → 在线支付 → 第三方揽派

设计目标：为 C 端用户提供简洁流畅的小程序体验，为运营人员提供清晰高效的 PC 后台管理界面。

---

## 二、设计风格定位

- **整体调性**：简洁现代、温暖亲切的小程序风格，弱化工具感，强化服务感
- **布局方式**：卡片式布局，大圆角，柔和阴影，留白充足
- **主色**：`#FF7D00`（温暖橙，用于主按钮、选中态、关键强调）
- **背景色**：`#F9F6F0`（暖米白，页面底色）
- **情感关键词**：温暖、可信、高效、轻松

## 四、组件规范

### 4.1 底部导航栏（tabBar）

- 三个入口：**首页**、**订单**、**我的**
- 图标 + 文字，未选中态灰色 `#999`，**选中态主色 `#FF7D00`**
- 高度约 100rpx（小程序）/ 56px（后台侧边栏对应概念），顶部细分割线

### 4.2 订单状态标签组件

| 状态 | 颜色 | 样式 |
|------|------|------|
| 待支付 | 橙色 `#FF7D00` | 浅橙底 `#FFF3E6` + 橙字 |
| 处理中 | 蓝色 `#3B82F6` | 浅蓝底 `#E8F1FF` + 蓝字 |
| 已完成 | 绿色 `#22C55E` | 浅绿底 `#E7F8EE` + 绿字 |
| 已取消 | 灰色 `#9CA3AF` | 浅灰底 `#F3F4F6` + 灰字 |

### 4.3 空状态组件

- 居中插画（简约线性图标）+ 提示文案（如"暂无订单"）+ 可选操作按钮
- 文案颜色 `#9CA3AF`，按钮为主色描边或填充

### 4.4 文件上传组件

- 虚线边框拖拽/点击上传区，内含上传图标 + "点击或拖拽文件到此处"
- 文件列表项：文件图标 + 文件名 + 大小 + 上传进度条（主色）+ 删除按钮
- 支持多文件，进度条完成变绿

### 4.5 价格计算实时预览组件

- 常驻于配置/确认页底部或侧边
- 实时展示：单价 × 数量 = 小计，含装订/增值/快递分项，总价大字强调
- 价格数字使用主色或深色加粗，分项灰色小字

## 五、交互规范

### 5.1 微信支付流程

- 点击"去支付" → 调起微信支付弹层（金额醒目）→ 支付成功 → 订单状态变更 + Toast 提示 → 跳转订单详情
- 支付失败/取消 → 保留订单于"待支付"，可重新支付

### 5.2 文件上传进度

- 上传中显示百分比进度条，禁止重复提交
- 完成后文件卡片显示"已上传"勾选标记

### 5.3 表单验证反馈

- 必填项为空时输入框红框 + 下方红字提示
- 手机号/重量等格式错误即时校验
- 提交按钮在验证通过前置灰

### 5.4 页面转场动画

- 页面间 push/pop 使用系统默认左右滑动
- 弹窗/底部弹层使用自下而上滑入 + 背景遮罩淡入
- 列表刷新使用下拉 loading 动画
