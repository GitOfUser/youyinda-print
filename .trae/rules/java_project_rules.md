### 一、项目基础信息
1. 项目名称：优印达
2. 产品形态：微信小程序 + Spring Boot 后端服务 + Vue3 运营管理后台
3. 核心业务：线上云打印（1:1对标小猴云印交互与功能）+ 低价快递寄件聚合平台
4. 核心模式：纯平台化聚合，所有打印生产、快递揽派履约均对接第三方服务商API，平台仅做流量分发、价格封装、订单中转、盈利差价赚取
5. 盈利核心：第三方基础服务价叠加自定义可配置盈利规则，形成平台最终展示售价，所有规则支持后台热更新，绝对禁止硬编码

### 二、技术栈强制规范（Java后端核心）
1. 基础框架：Spring Boot 2.7.18（稳定兼容版），打包方式jar，JDK版本1.8
2. 项目构建：Maven，pom.xml依赖必须完整可直接编译，无冗余依赖
3. ORM框架：MyBatis-Plus 3.5.3.1，简化CRUD操作，分页插件内置
4. 数据库：MySQL 8.0，字符集utf8mb4，所有表必须包含：
   - id（bigint 自增主键）
   - create_time（datetime 创建时间）
   - update_time（datetime 更新时间，自动更新）
   - is_delete（tinyint 软删除，0=未删除，1=已删除）
5. 缓存：Redis 6.x，用于用户登录态、接口限流、第三方基础价缓存、热点数据缓存
6. 接口规范：RESTful API风格，所有接口统一返回格式，路径前缀 /api/v1/ 区分版本
7. 安全鉴权：Spring Security + JWT 实现接口鉴权，小程序端基于微信session_key实现登录态管理，用户敏感信息AES加密存储，所有接口必须做参数校验
8. 第三方对接：
   - HTTP客户端：OpenFeign 做统一接口封装，配合Resilience4j实现超时重试、熔断降级
   - 微信生态：使用微信官方Java V3 SDK对接微信支付、小程序授权
   - 所有第三方接口配置（url、appId、密钥、超时时间）必须抽离到application.yml配置文件，禁止硬编码
9. 代码规范：
   - 严格遵循阿里巴巴Java开发规范，分层架构清晰：controller → service → serviceImpl → mapper → entity
   - 扩展目录：config（配置类）、common（公共类）、util（工具类）、dto（入参）、vo（出参）、enums（枚举）、exception（异常处理）
   - 所有类、方法、核心逻辑必须添加中文注释，命名采用驼峰命名法，见名知意
   - 事务管理：核心订单流程必须添加@Transactional事务注解，rollbackFor = Exception.class
10. 异常处理：全局统一异常处理器，自定义业务异常类，错误码体系完整，所有异常全日志记录

### 三、开发约束规则
1. 所有功能开发前，必须先确认业务逻辑无歧义，再生成代码，禁止擅自修改核心业务流程
2. 每次开发只处理一个独立模块，开发完成后必须说明生成/修改的文件清单、核心代码功能
3. 严格遵循微信小程序平台规范，兼容微信生态所有能力（微信支付、地址授权、消息推送、聊天文件上传）
4. 价格计算逻辑必须无漏洞，所有计价公式完全对用户透明，无隐藏消费，计算结果保留2位小数
5. 第三方接口对接必须实现：配置化管理、超时重试、熔断降级、全链路日志追踪、异常兜底处理
6. 订单状态流转必须有完整的日志记录，状态变更必须做前置校验，禁止非法状态跳转
