# 优印达项目 - 前后端联调指南

## 联调准备

### 1. 环境准备
- **后端环境**：
  - JDK 1.8+
  - Maven 3.6+
  - MySQL 8.0+
  - Redis 6.x（可选）

- **前端环境**：
  - 微信开发者工具
  - 微信小程序AppID（测试号即可）

### 2. 数据库初始化
```bash
# 连接MySQL
mysql -u root -p

# 创建数据库并导入数据
source d:\Projects\youyinda-print\youyinda-backend\src\main\resources\init_test.sql
```

### 3. 后端启动
```bash
# 进入后端目录
cd d:\Projects\youyinda-print\youyinda-backend

# 编译项目
mvn clean package -DskipTests

# 启动服务
java -jar target/youyinda-backend-1.0.0.jar
```

### 4. 前端配置
修改 `youyinda-miniprogram/utils/request.js` 中的BASE_URL：
```javascript
const BASE_URL = 'http://localhost:8080/api/v1';
```

## 常见问题处理

### 1. 跨域问题
**问题**：前端请求后端接口报跨域错误
**解决**：检查后端WebMvcConfig配置

### 2. Token过期
**问题**：登录后一段时间后操作失败
**解决**：检查Token有效期配置，实现自动刷新

### 3. 接口超时
**问题**：某些接口请求超时
**解决**：检查网络连接，调整超时时间

### 4. 数据格式错误
**问题**：前后端数据格式不一致
**解决**：统一数据格式，添加类型转换

### 5. 权限问题
**问题**：某些接口返回401或403
**解决**：检查Token是否正确，检查权限配置
